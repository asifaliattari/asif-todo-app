## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+
- Python 3.13+
- Neon PostgreSQL account (free tier)
- uv (Python package manager)

### Setup

1. **Clone the repository**
   ```bash
   cd "D:/hakathon/asif_todo_app_phase2"
   ```

2. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   cp .env.example .env.local
   # Edit .env.local and add your API URL
   npm run dev  # http://localhost:3000
   ```

3. **Backend Setup**
   ```bash
   cd backend
   uv sync
   cp .env.example .env
   # Edit .env and add your DATABASE_URL and JWT secret
   uv run uvicorn main:app --reload --port 8000
   ```

4. **Database Setup**
   - Sign up at [Neon](https://neon.tech)
   - Create a new project
   - Copy the connection string
   - Add to `backend/.env` as `DATABASE_URL`
   - Tables will be created automatically on first run

## ğŸ” Authentication Flow

1. User signs up or logs in
2. Backend generates JWT token
3. Frontend stores token in localStorage
4. All API requests include `Authorization: Bearer <token>` header
5. Backend verifies token and extracts user_id
6. Data is filtered by user_id (user isolation)

## ğŸ¨ API Endpoints

### Authentication
- `POST /api/auth/signup` - Create account
- `POST /api/auth/login` - Login

### Tasks
- `GET /api/tasks` - Get all user's tasks
- `POST /api/tasks` - Create new task
- `PUT /api/tasks/{id}` - Update task
- `PATCH /api/tasks/{id}/complete` - Toggle completion
- `DELETE /api/tasks/{id}` - Delete task

See `@specs/api/rest-endpoints.md` for detailed documentation.
